📖 Sobre o Projeto

A Vinharia Agnello é uma aplicação desenvolvida para realizar o cadastro e gerenciamento de vinhos de forma simples e interativa.

O sistema permite cadastrar diferentes rótulos de vinho, armazenando informações importantes como:

Nome do vinho
Tipo do vinho
Safra
Quantidade em estoque

Além disso, o projeto realiza validações automáticas, classificação dos vinhos conforme a safra e geração de relatórios utilizando funções de alta ordem do JavaScript.

O principal objetivo do projeto é praticar conceitos fundamentais de desenvolvimento web e lógica de programação utilizando JavaScript.

🧠 Funções de Alta Ordem Utilizadas

O projeto utiliza funções de alta ordem do JavaScript para manipular os dados dos vinhos.

🔹 forEach()

Utilizado para listar todos os vinhos cadastrados no console.

🔹 filter()

Utilizado para filtrar vinhos com estoque abaixo de 5 unidades.

🔹 map()

Utilizado para transformar os nomes dos vinhos em letras maiúsculas.

🔹 reduce()

Utilizado para calcular o estoque total da vinícola.

📦 Controle de Estoque

O sistema verifica automaticamente a quantidade disponível em estoque.

Caso o estoque esteja baixo, o vinho é exibido na lista de estoque baixo no console.

📊 Relatório Final

Ao final dos cadastros, o sistema apresenta:

Quantidade total de vinhos cadastrados
Lista completa dos vinhos
Vinhos com estoque baixo
Nomes dos vinhos em caixa alta
Estoque total da vinícola
